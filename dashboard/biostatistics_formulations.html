Core thesis: Spatial imputation methods introduce spatially correlated reconstruction errors that inflate cell–cell interaction statistics, generating false biological interactions. This framework detects, corrects, and quantifies this artifact.

1. Poisson Measurement Model

Single-Cell Observation Model

For cell $i$ at spatial location $\mathbf{s}_i \in \mathbb{R}^2$ and gene $g$:

$$y_{ig} \sim \text{Poisson}(\delta_g \cdot \lambda_{ig}), \quad g \in S \subset \{1, \dots, G\}$$
SymbolMeaningTypical Value
$y_{ig}$Observed transcript count$\{0, 1, 2, \dots\}$
$\lambda_{ig}$True expression rate$> 0$
$\delta_g$Detection efficiency (probe-specific)$(0, 1]$
$S$Measured gene panel$|S| = m \ll G$
$G$Total genes in genome$\approx 20{,}000$

Key Properties

Mean-Variance Relationship
$$\mathbb{E}[y_{ig}] = \text{Var}(y_{ig}) = \delta_g \lambda_{ig}$$

Signal-to-noise ratio:

$$\text{SNR}_{ig} = \frac{(\delta_g \lambda_{ig})^2}{\delta_g \lambda_{ig}} = \delta_g \lambda_{ig}$$
Biological implication: SNR scales linearly with expression level. Low-expression genes (e.g., transcription factors, rare cytokines) have inherently worse SNR — this is physics, not a platform defect.

The Selection Matrix

Panel Observation Operator
$$\mathbf{y}_{i,\text{obs}} = A_S \, \mathbf{y}_i \in \mathbb{Z}_+^m$$

where $A_S \in \{0,1\}^{m \times G}$ selects rows of the identity matrix corresponding to panel $S$:

$$(A_S)_{jg} = \begin{cases} 1 & \text{if gene } g \text{ is the } j\text{-th gene in panel } S \\ 0 & \text{otherwise} \end{cases}$$

Observation fraction: $\rho = m/G$. For Xenium 313-gene panel: $\rho \approx 0.016$ (1.6%).

2. Low-Rank Structure

Gene Program Decomposition
$$\log \lambda_{ig} = \mu_g + \sum_{k=1}^{K} u_{ik} \, v_{gk} + \epsilon_{ig}$$

In matrix form:

$$\log \boldsymbol{\Lambda} = \mathbf{1}\boldsymbol{\mu}^\top + \mathbf{U}\mathbf{V}^\top + \mathbf{E}$$
SymbolDimensionMeaning
$\boldsymbol{\mu}$$G \times 1$Baseline log-expression per gene
$\mathbf{U}$$N \times K$Cell loading matrix (program activities)
$\mathbf{V}$$G \times K$Gene program matrix (gene membership)
$K$scalarNumber of gene programs ($K \ll \min(N,G)$)
$\mathbf{E}$$N \times G$Residual noise

Why Recovery Is Possible

Degrees of Freedom Argument

Without structure: $N \times G$ unknowns, $N \times m$ observations. Ratio $= G/m \approx 64\times$ underdetermined.

With rank $K$: Parameters $= G + K(N + G)$. Recovery requires:

$$m > \frac{G}{N} + K\left(1 + \frac{G}{N}\right)$$

For $N = 100{,}000$, $G = 20{,}000$, $K = 30$:

$$m > 0.2 + 36.0 = 36.2$$
Key insight: You need roughly $m > K$ measured genes. With ~30 gene programs, even 50 genes is theoretically sufficient. This is why imputation works at all.

3. Recovery Guarantees

Poisson Matrix Completion Bound (Cao & Xie, 2015)

Under rank-$K$ structure, incoherence, and uniform panel selection:

$$\boxed{\frac{1}{NG}\|\hat{\boldsymbol{\Lambda}} - \boldsymbol{\Lambda}\|_F^2 \leq C \cdot \frac{K(N+G)\,\lambda_{\max}}{Nm}}$$

For $N \gg G$, simplifies to:

$$\text{MSE} \approx C \cdot \frac{K \, \lambda_{\max}}{m}$$
TermEffectMeaning
$K$ ↑Error ↑More gene programs = harder recovery
$\lambda_{\max}$ ↑Error ↑Poisson variance scales with rate
$m$ ↑Error ↓Larger panel = better recovery
$N$ ↑Error ↓More cells help learn gene programs
Critical ratio: $K/m$ — rank divided by panel size governs reconstruction quality. If $K/m \to 1$, recovery degrades catastrophically.

Penalized Maximum Likelihood Estimator

Nuclear Norm Regularized Estimation
$$\hat{\boldsymbol{\Lambda}} = \arg\min_{\boldsymbol{\Lambda} \succ 0} \left[ -\ell(\boldsymbol{\Lambda}) + \tau \|\boldsymbol{\Lambda}\|_* \right]$$

Log-likelihood:

$$\ell(\boldsymbol{\Lambda}) = \sum_{i=1}^N \sum_{g \in S} \left[ y_{ig} \log \lambda_{ig} - \lambda_{ig} - \log(y_{ig}!) \right]$$

$\|\boldsymbol{\Lambda}\|_* = \sum_k \sigma_k(\boldsymbol{\Lambda})$ is the nuclear norm (convex surrogate for rank).

4. Per-Gene Reconstruction Error

Gene-Specific MSE Decomposition
$$\text{MSE}(g) = \underbrace{\frac{1}{\delta_g \bar{\lambda}_g}}_{\text{Poisson shot noise}} + \underbrace{\|\mathbf{v}_g\|^2(1 - R_g^2)}_{\text{Panel incompleteness}}$$

Gene Predictability ($R^2_g$)

Latent-Space Coefficient of Determination
$$R_g^2 = \frac{\mathbf{v}_g^\top \mathbf{V}_S^\top (\mathbf{V}_S \mathbf{V}_S^\top)^{-1} \mathbf{V}_S \, \mathbf{v}_g}{\|\mathbf{v}_g\|^2}$$

Residual variance:

$$\sigma_g^2 = \|\mathbf{v}_g\|^2 (1 - R_g^2)$$
$R_g^2$ ValueMeaningRecovery Quality
$\approx 1$Gene lies in span of measured genesReliable
$0.5 - 0.8$Partially predictableModerate
$\approx 0$Orthogonal to measured panelUnrecoverable

5. Moran's I (Global Spatial Autocorrelation)

Spatial Statistic Used in Regression Calibration

Question

"Are nearby cells more similar in expression than expected by chance?"

Moran's I
$$I = \frac{N}{W} \cdot \frac{\displaystyle\sum_{i=1}^{N} \sum_{j=1}^{N} w_{ij}(x_i - \bar{x})(x_j - \bar{x})}{\displaystyle\sum_{i=1}^{N} (x_i - \bar{x})^2}$$
SymbolMeaning
$N$Number of cells
$x_i$Expression of the gene in cell $i$
$\bar{x}$Mean expression across all cells
$w_{ij}$Spatial weight: 1 if $i,j$ are neighbors, 0 otherwise
$W = \sum_{i}\sum_{j} w_{ij}$Total weight (2 × number of edges)

6. Geary's C (Local Dissimilarity)

Spatial Statistic Detects Local Heterogeneity

Question

"How different are neighboring cells from each other?"

$$NE(a, b) = \frac{\text{observed}(a,b) - \mathbb{E}_{\text{perm}}[\text{count}(a,b)]}{\text{Std}_{\text{perm}}[\text{count}(a,b)]}$$

$\mathbb{E}_{\text{perm}}$Expected count under random label permutations
$\text{Std}_{\text{perm}}$Standard deviation under permutations

9. Cell-Cell Interaction Test Statistic

CCI Score for Ligand-Receptor Pair $(l, r)$
$$T_{lr} = \frac{1}{|\mathcal{E}|} \sum_{(i,j) \in \mathcal{E}} x_{il} \cdot x_{jr}$$

where $\mathcal{E}$ is the set of spatially adjacent cell pairs (edges in the spatial graph).

Significance Testing

Permutation Test

Under the null hypothesis of no spatial interaction:

  1. Randomly permute cell labels: $\pi \sim \text{Uniform}(S_N)$
  2. Compute null statistic: $T_{lr}^{(\pi)} = \frac{1}{|\mathcal{E}|} \sum_{(i,j) \in \mathcal{E}} x_{\pi(i),l} \cdot x_{j,r}$
  3. Repeat $B$ times
  4. $p\text{-value} = \frac{\#\{T_{lr}^{(\pi)} \geq T_{lr}\} + 1}{B + 1}$

Multiple testing correction via Benjamini-Hochberg: reject if $p_{\text{adj}} < 0.05$.

10. Bias Decomposition (The Core Insight)

Why Imputation Inflates CCI

If the imputed expression is $\hat{\lambda}_{ig} = \lambda_{ig} + e_{ig}$, then:

$$\mathbb{E}[\hat{\lambda}_{il} \cdot \hat{\lambda}_{jr}] = \underbrace{\lambda_{il} \cdot \lambda_{jr}}_{\text{true signal}} + \underbrace{\lambda_{il} \cdot \mathbb{E}[e_{jr}] + \lambda_{jr} \cdot \mathbb{E}[e_{il}]}_{\text{= 0 if unbiased}} + \underbrace{\text{Cov}(e_{il}, e_{jr})}_{\text{BIAS}}$$
The problem: For graph-based imputation methods (GNN, diffusion, any method using spatial neighbors), $\text{Cov}(e_{il}, e_{jr}) > 0$ when cells $i$ and $j$ are neighbors. The imputation algorithm shares information across spatial edges — the exact same edges used to compute CCI. This creates a systematic positive bias in the CCI statistic.
CCI Bias Decomposition
$$\mathbb{E}[\hat{T}_{lr}] = T_{lr}^{\text{true}} + \underbrace{\frac{1}{|\mathcal{E}|}\sum_{(i,j) \in \mathcal{E}} \text{Cov}(e_{il}, e_{jr})}_{\text{bias} > 0 \text{ for graph-based methods}}$$

11. Regression Calibration

Core Contribution Paper Lead

Bias Estimation via Moran's I

Estimate spatial error correlation for each gene:

$$\hat{\rho}_g = I_g = \frac{N}{W} \cdot \frac{\sum_{i,j} w_{ij}(\hat{\lambda}_{ig} - \bar{\hat{\lambda}}_g)(\hat{\lambda}_{jg} - \bar{\hat{\lambda}}_g)}{\sum_i (\hat{\lambda}_{ig} - \bar{\hat{\lambda}}_g)^2}$$

Cross-gene spatial error correlation:

$$\hat{\rho}_{lr} = \sqrt{|\hat{\rho}_l| \cdot |\hat{\rho}_r|}$$

Bias estimate:

$$\widehat{\text{bias}}_{lr} = \hat{\rho}_{lr} \cdot \sqrt{\text{MSE}(l) \cdot \text{MSE}(r)}$$
Corrected CCI Statistic
$$\hat{T}_{lr}^{\text{corrected}} = \hat{T}_{lr}^{\text{naive}} - \widehat{\text{bias}}_{lr}$$

Corrected variance:

$$\text{Var}(\hat{T}_{lr}^{\text{corrected}}) \approx \text{Var}(\hat{T}_{lr}^{\text{naive}}) + \frac{\text{MSE}(l) \cdot \text{MSE}(r)}{|\mathcal{E}|}$$

12. Corrected Inference Procedure

Complete Algorithm
  1. Compute naive CCI: $\hat{T}_{lr}$ on imputed data
  2. Estimate per-gene MSE: $\widehat{\text{MSE}}(g)$ via cross-validation on measured genes
  3. Compute Moran's I for ligand and receptor: $\hat{\rho}_l, \hat{\rho}_r$
  4. Estimate bias: $\widehat{\text{bias}}_{lr} = \sqrt{|\hat{\rho}_l||\hat{\rho}_r|} \cdot \sqrt{\widehat{\text{MSE}}(l) \cdot \widehat{\text{MSE}}(r)}$
  5. Correct: $\hat{T}_{lr}^{\text{corr}} = \hat{T}_{lr} - \widehat{\text{bias}}_{lr}$
  6. Permutation test on corrected statistic (subtract same bias from null)
  7. BH correction for multiple testing

13. Error Propagation: Cell Type Classification

Misclassification Bound

For cell $i$ of true type $t$, classified via nearest centroid on reconstructed expression:

$$P(\text{misclassify cell } i) \leq \exp\left(-\frac{\Delta_t^2}{8 \cdot \sigma_{\text{recon}}^2}\right)$$

where:

$$\Delta_t = \min_{t' \neq t} \|\boldsymbol{\mu}^{(t)} - \boldsymbol{\mu}^{(t')}\|_2 \quad \text{(minimum inter-type distance)}$$ $$\sigma_{\text{recon}}^2 = \sum_{g=1}^{G} \text{MSE}(g) \quad \text{(total reconstruction variance)}$$
Prediction: Cell types with small $\Delta_t$ (e.g., CD4 naive vs CD4 memory T cells) fail first as panel size $m$ decreases. The critical panel size for type $t$: $$m_t^* \approx \frac{8 \, K \, \lambda_{\max}}{\Delta_t^2}$$

14. Error Propagation: CCI False Positive Rate

Variance Inflation of CCI Statistic
$$\text{Var}(\hat{T}_{lr}) \geq \frac{1}{|\mathcal{E}|}\left[\bar{\lambda}_r^2 \cdot \text{MSE}(l) + \bar{\lambda}_l^2 \cdot \text{MSE}(r) + \text{MSE}(l) \cdot \text{MSE}(r)\right]$$
ScenarioVariance ScalingReliability
Both $l, r$ measured ($\in S$)Poisson noise onlyHigh
One imputed, one measured$\propto \text{MSE}(\text{imputed gene})$Moderate
Both $l, r$ imputed ($\notin S$)$\propto \text{MSE}(l) \times \text{MSE}(r)$Low (quadratic)
Quadratic degradation: CCI involving two imputed genes is quadratically less reliable than CCI with one measured gene. This is the strongest argument for panel-aware CCI analysis.

15. Error Propagation: Spatial Domain Detection

Spatial Domain Stability

Spatial domain detection typically uses Leiden/Louvain clustering on a spatial + expression graph. The clustering output depends on edge weights:

$$w_{ij}^{\text{eff}} = \text{sim}(\hat{\boldsymbol{\lambda}}_i, \hat{\boldsymbol{\lambda}}_j) \cdot \text{spatial\_weight}(i,j)$$

Reconstruction error perturbs the similarity:

$$\text{sim}(\hat{\boldsymbol{\lambda}}_i, \hat{\boldsymbol{\lambda}}_j) = \text{sim}(\boldsymbol{\lambda}_i, \boldsymbol{\lambda}_j) + O\left(\frac{\sigma_{\text{recon}}}{\Delta_{\min}}\right)$$

Domains merge or split when $\sigma_{\text{recon}} / \Delta_{\min} \gtrsim 1$.

16. Panel Perturbation Stability Analysis

NOT a Bootstrap Stability Measure

Procedure
  1. From panel $S$ ($m$ genes), create $B$ sub-panels $S_1^*, \dots, S_B^*$ by dropping fraction $\alpha$ of genes
  2. For each $S_b^*$: re-impute $\hat{\boldsymbol{\Lambda}}_b$, re-compute CCI $\hat{T}_{lr}^{(b)}$
  3. Compute per-pair stability metrics
Stability Metrics

Coefficient of Variation

$$\text{CV}_{lr} = \frac{\text{Std}_b[\hat{T}_{lr}^{(b)}]}{|\text{Mean}_b[\hat{T}_{lr}^{(b)}]| + \epsilon}$$

Stability Score

$$\text{Stability}_{lr} = 1 - \text{clip}(\text{CV}_{lr}, 0, 1)$$

Robustness

$$\text{Robustness}_{lr} = \frac{1}{B}\sum_{b=1}^{B} \mathbb{1}[\hat{T}_{lr}^{(b)} > 0]$$
StabilityMeaningAction
$\approx 1.0$Robust to panel compositionTrust
$\approx 0.5$Fragile — depends on which genes are includedFlag
$\approx 0.0$Unstable — conclusion is noiseDiscard

17. Calibration Analysis

Calibration Plot Construction
  1. Bin significant CCI pairs by predicted reliability (stability score)
  2. In each bin $b$, compute False Discovery Proportion:
$$\text{FDP}(b) = 1 - \frac{\text{\# true positives in bin } b}{\text{\# total in bin } b}$$

Perfect calibration: FDP decreases monotonically as reliability increases.

Target: Spearman correlation between predicted reliability and actual error $\geq 0.7$. FDP in highest reliability bin $< 0.1$. FDP in lowest reliability bin $> 0.5$.

22. Complete Model Summary

The Entire Framework in One Box
$$\boxed{\begin{aligned} &\textbf{Observation:} && y_{ig} \sim \text{Poisson}(\delta_g \lambda_{ig}), \quad g \in S,\; |S| = m \\[6pt] &\textbf{Structure:} && \log \lambda_{ig} = \mu_g + \sum_{k=1}^K u_{ik}\,v_{gk} + \epsilon_{ig} \\[6pt] &\textbf{Recovery MSE:} && \frac{1}{NG}\|\hat{\boldsymbol{\Lambda}} - \boldsymbol{\Lambda}\|_F^2 \leq C\,\frac{K\lambda_{\max}}{m} \\[6pt] &\textbf{Per-gene error:} && \text{MSE}(g) = \frac{1}{\delta_g \bar{\lambda}_g} + \|\mathbf{v}_g\|^2(1-R_g^2) \\[6pt] &\textbf{CCI bias:} && \mathbb{E}[\hat{T}_{lr}] = T_{lr}^{\text{true}} + \rho_{lr}\sqrt{\text{MSE}(l)\,\text{MSE}(r)} \\[6pt] &\textbf{Corrected CCI:} && \hat{T}_{lr}^{\text{corr}} = \hat{T}_{lr} - \hat{\rho}_{lr}\sqrt{\widehat{\text{MSE}}(l)\,\widehat{\text{MSE}}(r)} \\[6pt] &\textbf{Cell typing:} && P(\text{misclassify}) \leq \exp\!\left(-\frac{\Delta_t^2}{8\sum_g \text{MSE}(g)}\right) \\[6pt] &\textbf{CCI variance:} && \text{Var}(\hat{T}_{lr}) \propto \text{MSE}(l)\cdot\text{MSE}(r) \;\text{ if both imputed} \end{aligned}}$$

Three governing quantities:

$K/m$Rank over panel size — can you capture the latent structure?
$R_g^2$Per-gene predictability — is this gene in the measured subspace?
$\Delta_t$ / effect sizeBiological signal strength — can biology survive the noise?
