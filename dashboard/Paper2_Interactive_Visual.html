1The Problem Statement

The One-Paragraph Paper Summary

In resected non-small cell lung cancer patients, when a tumor-immune neighborhood composition score $W_i$ is derived from 10x Visium spatial transcriptomics data and entered as a continuous covariate in a Cox proportional hazards model for overall survival, the partial likelihood estimator of the log-hazard ratio $\beta$ is biased toward the null because $W_i$ is a noisy estimate of a latent spatial feature $X_i$, and the noise variance is heteroscedastic — it varies across patients as a function of tissue spot count $m_i$ and spatial autocorrelation $\rho_i$. We develop a generalized regression calibration method with patient-specific reliability ratios $\lambda_i$ and demonstrate via simulation that it reduces bias by 85–95%, restores confidence interval coverage to near-nominal (93–95%), and recovers 70–80% of the power lost to measurement error.

4The Measurement Error Model

In reality, we never observe $X_i$. We observe $W_i$, a noisy version.

The Structural Measurement Error Model
\begin{align} X_i &\sim N(\mu_X, \sigma_X^2) \tag{latent biomarker} \\[4pt] U_i \mid X_i, m_i, \rho_i &\sim N(0, \sigma_i^2) \tag{measurement error} \\[4pt] W_i &= X_i + U_i \tag{observed score} \\[4pt] \sigma_i^2 &= \alpha_0 + \frac{\alpha_1}{m_i} + \alpha_2 \rho_i \tag{heteroscedastic variance} \end{align}
Each patient's tumor is like a landscape. $X_i$ is the true elevation. You drop $m_i$ measuring sticks and average. Smooth terrain (low $\rho_i$) = accurate average. Rugged terrain (high $\rho_i$) = nearby sticks give correlated readings = fewer independent measurements = noisier average.

5Attenuation Bias

The Fundamental Problem

Using $W_i$ instead of $X_i$ in Cox regression, the estimated coefficient converges to:

$$\tilde{\beta} \xrightarrow{p} \beta \cdot \lambda, \qquad \lambda = \frac{\sigma_X^2}{\sigma_X^2 + \sigma_U^2} < 1$$

Since $\lambda < 1$, $|\tilde{\beta}| < |\beta|$. The estimate is shrunk toward zero. Your biomarker looks weaker than it is.

Algebraic Derivation

Why Attenuation Happens — Step by Step

In linear regression with $Y = \beta X + \epsilon$ and $W = X + U$:

\begin{align} \tilde{\beta} &= \frac{\text{Cov}(W, Y)}{\text{Var}(W)} \\[8pt] &= \frac{\text{Cov}(X+U, \; \beta X + \epsilon)}{\text{Var}(X+U)} \\[8pt] &= \frac{\beta \cdot \text{Var}(X) + \cancelto{0}{\text{Cov}(U,\beta X)} + \cancelto{0}{\text{Cov}(X,\epsilon)} + \cancelto{0}{\text{Cov}(U,\epsilon)}}{\text{Var}(X) + \text{Var}(U)} \\[8pt] &= \beta \cdot \frac{\text{Var}(X)}{\text{Var}(X) + \text{Var}(U)} = \beta \cdot \lambda \end{align}

9Estimating $\sigma_i^2$ (Bootstrap)

Spot-Level Bootstrap Algorithm

For each patient $i$ with $m_i$ spots:

  1. Subsample 80% of spots ($B = 200$ times)
  2. Reconstruct spatial graph on each subsample
  3. Recompute neighborhood score $W_i^{(b)}$
  4. $\hat{\sigma}_i^2 = \text{Var}(W_i^{(1)}, \ldots, W_i^{(B)}) \times \frac{0.80}{1 - 0.80}$

Then fit parametric model: $\hat{\sigma}_i^2 = \hat{\alpha}_0 + \hat{\alpha}_1/m_i + \hat{\alpha}_2\rho_i$

10Estimating $\sigma_X^2$ (Method of Moments)

Identification Formula
$$\text{Var}(W_i) = \sigma_X^2 + \mathbb{E}[\sigma_i^2] \quad \Longrightarrow \quad \boxed{\hat{\sigma}_X^2 = \max\left\{\widehat{\text{Var}}(W_i) - \frac{1}{n}\sum\hat{\sigma}_i^2, \; 0\right\}}$$

Our example: $\widehat{\text{Var}}(W) = 0.0486$, $\bar{\hat{\sigma}}^2 = 0.0208$, so $\hat{\sigma}_X^2 = 0.0278$.

11The Regression Calibration Correction

The Core Formula
$$\boxed{\hat{X}_i = \hat{\mu}_W + \hat{\lambda}_i (W_i - \hat{\mu}_W)}$$
  • If $\hat{\lambda}_i \approx 1$ (precise measurement): $\hat{X}_i \approx W_i$ — trust the observation
  • If $\hat{\lambda}_i \approx 0$ (noisy measurement): $\hat{X}_i \approx \hat{\mu}_W$ — trust the population mean

14Variance Estimation

Two Sources of Uncertainty
$$\widehat{\text{Var}}(\hat{\beta}_{\text{RC}}) = \underbrace{I^{-1}(\hat{\beta}_{\text{RC}})}_{\text{Cox sampling}} + \underbrace{\mathbf{D}^\top \boldsymbol{\Sigma}_\alpha \mathbf{D}}_{\text{from estimating } \hat{\alpha}}$$

In practice: nonparametric bootstrap — resample patients, re-estimate everything, take bootstrap variance.

17Complete Model Summary

The Entire Framework in One Box
$$\boxed{\begin{aligned} &X_i \sim N(\mu_X, \sigma_X^2) && \text{latent, graph-conditional} \\[4pt] &W_i = X_i + U_i && \text{observed score} \\[4pt] &U_i \sim N(0, \sigma_i^2), \;\; \sigma_i^2 = \alpha_0 + \frac{\alpha_1}{m_i} + \alpha_2\rho_i && \text{heteroscedastic error} \\[4pt] &\hat{\sigma}_X^2 = \max\{\widehat{\text{Var}}(W) - \bar{\hat{\sigma}}^2, 0\} && \text{identification} \\[4pt] &\lambda_i = \sigma_X^2 / (\sigma_X^2 + \sigma_i^2) && \text{patient-specific reliability} \\[4pt] &\hat{X}_i = \hat{\mu}_W + \hat{\lambda}_i(W_i - \hat{\mu}_W) && \text{calibrated covariate} \\[4pt] &h(t|\hat{X}_i) = h_0(t)\exp(\hat{\beta}_{\text{RC}}\hat{X}_i) && \text{corrected Cox} \end{aligned}}$$
